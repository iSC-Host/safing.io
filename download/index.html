Limited Support
The following platforms currently only receive limited support. This means that not all features may be available and especially UI components may be impaired or completely broken. Additionally, issues with these platforms are handled with a lower priority. We provide workarounds where we can.
While we aim to fully support these platforms as soon as we can, we do not have the resources to do so at the moment.
If your platform is not listed, or you think that it is underrepresented, please contact us!

Windows 7, 8

Core fully functional and supported.
Notifications and Prompts are not implemented. For the best experience, do not use the Prompt flag until a workaround is available.

Linux 2.4+

Core fully functional and supported.
Desktop environment integration varies heavily. Currently only packaged for Debian and it's derivatives.
Manual installation instructions.
